#### Tensor Operations
- In PyTorch, pointers to allocated memory
- Long 1d array
- `x.stride(0) = 4` to go down rows with 4 cols, `x.stride(1) = 1` to move to next element in row
- Something like `y=x[0]` will not create a copy, just a view, this helps conserve storage AND compute
- Same for mutation - i.e. `x[0][0] = 100` will lead to `y[0][0] = 100`
- Non contiguous views can't have views created on them, for e.g. transpose `y = x. transpose(1, 0). contiguous (). view(2, 3)]`
- Element-wise operations like x.pow(2) etc.
